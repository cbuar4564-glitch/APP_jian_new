# APP_jian_new — 생활 불편 해소 앱 모음

사람들이 생활 속에서 겪는 불편을 빠르게 덜어주는 작은 웹 앱 모음입니다.
설치·서버·빌드가 필요 없고, **모든 처리가 사용자 기기(브라우저) 안에서만** 이뤄집니다.

## 들어있는 앱

### 💰 내돈지킴이 — [`money-guardian/`](money-guardian/index.html)
구독 해지·환불·소비자 항의 문구를 자동으로 만들어주고, 돈을 참고용으로 계산해 줍니다.
- 구독 해지 / 환불 요청 / 항의·신고 문구 자동 작성 (정중·단호 톤)
- 내돈계산기: 구독 일할 환불 추정 / 표시가–실제청구 차액 / 위약금 과다 점검 (모두 참고용)

### 🌙 꿈동산 — [`bedtime/`](bedtime/index.html)
지친 부모를 위한 아이 잠자리 도우미.
- 아이 이름과 오늘의 마음(무서운 꿈·다툼·걱정·그리움 등)에 맞춘 잠자리 이야기 생성
- 기기 내장 음성으로 **차분하게 읽어주기**
- 잠자리 루틴 체크리스트 + 1분 "함께 숨쉬기" 가이드

### 🍽️ 결정 룰렛 — [`roulette/`](roulette/index.html)
"오늘 뭐 먹지?" 결정 피로를 룰렛 한 번으로 해결.
- 후보 직접 추가 + 빠른 프리셋(점심·저녁·카페·분식·야식·술안주·할 일·벌칙)
- 돌리는 룰렛 애니메이션으로 즉석 결정, 결과는 기기에 저장(localStorage)
- 100% 온디바이스·무료, 재미로 즐기는 도구

### 🎨 퍼스널컬러 옷장 — [`personal-color/`](personal-color/index.html)
셀카 한 장으로 나에게 어울리는 색을 찾아주는 도우미.
- 사진의 볼(뺨) 색을 분석해 봄/여름/가을/겨울 웜·쿨 타입 추정
- 어울리는 색 팔레트 + 피하면 좋은 색 + 스타일 팁, 팔레트 카드 이미지 저장
- **사진은 기기 안에서만 분석**(업로드 없음), 결과는 재미·참고용

### 🐾 펫 꾸미기 스튜디오 — [`pet-card/`](pet-card/index.html)
반려동물 사진을 SNOW처럼 귀엽게 꾸며 SNS에 자랑하는 도구.
- 📷 **카메라로 바로 찍기**(전/후면 전환·셀카 거울보정) 또는 앨범에서 고르기
- 🖼 **콜라주**: 한 장 / 2칸 / 3칸 / 4칸, 비율 4:5·1:1·9:16(스토리)
- 🎨 **스티커**(왕관·안경·하트·반짝·표정·간식·꽃·파티 등 90여 종) — 손으로 옮기고 **크기·회전·삭제**
- ✍️ **글자**: 내용 입력 + 글자색 자유 선택(외곽선으로 가독성)
- ✨ **필터** 7종(밝게·따뜻·시원·흑백·빈티지·선명) + PNG 저장·바로 공유
- 🔒 **카메라·사진·글자 모두 기기 안에서만** 처리, 어디에도 전송·저장되지 않음

### 🧠 퀴즈 한 입 — [`truth-quiz/`](truth-quiz/index.html)
주제를 골라 푸는 상식·공부 퀴즈 묶음. (OX / 4지선다)
- **묶음**: 생활 상식 오해 · 육아 상식 · 기초 영어 단어 · 기초 일본어 단어 (검증된 100+문제, 계속 확장)
- 한 판 10문제 → "나 N개 틀렸어!" 결과 공유, **다시 풀면 안 본 문제 우선**(반복 최소화)
- AI 생성 + **사실/번역 검증**을 거친 카드, 100% 온디바이스, 재미·참고용(건강·의학·육아는 전문가 상담)

## 사용 방법
```
index.html 을 더블클릭 → 홈에서 원하는 앱 선택
(또는 각 앱 폴더의 index.html 을 직접 열기)
```

## 개인정보 보호 (중요)
- 입력한 모든 내용은 **앱이 외부 서버·외부 AI API로 전송하지 않습니다.**
- 꿈동산의 읽어주기는 기기에 내장된 음성을 우선 사용합니다. (다만 일부 OS·브라우저는 음성 합성을 위해 텍스트를 자체적으로 처리할 수 있어, '절대 전송 없음'을 100% 보장하진 않습니다.)
- `.env`, `*.key`, `*.pem`, `secrets/`, `private/` 등 비밀/개인정보 파일은 `.gitignore`로 저장소에서 제외됩니다.
