# 내돈지킴이 (APP_jian_new)

생활 속 "돈 관련 불편"을 빠르게 해결하도록 돕는 가벼운 웹 앱입니다.
구독 해지·환불 요청·소비자 항의 문구를 자동으로 만들어주고, 환불/차액/위약금을 참고용으로 계산해 줍니다.

> 생활 불편 해소 AI 앱 프로젝트의 첫 번째 MVP입니다.

## 핵심 기능
- **구독 해지** 요청 문구 자동 작성
- **환불 요청** 문구 자동 작성
- **항의·신고** 문구 자동 작성 (1372 소비자상담센터 안내 포함)
- **내돈계산기** — 구독 일할 환불 추정 / 표시가–실제청구 차액 / 위약금 과다 점검 (모두 참고용)

말투(정중/단호)를 고르면 문장 톤이 바뀌고, 결과는 한 번에 복사할 수 있습니다.

## 사용 방법
설치·서버·빌드가 필요 없습니다.

```
index.html 파일을 더블클릭 → 브라우저에서 바로 실행
```

## 개인정보 보호 (중요)
- 입력한 모든 내용은 **사용자 기기(브라우저) 안에서만** 처리됩니다.
- 외부 서버·외부 AI API로 **아무것도 전송하지 않습니다.** (네트워크 통신 코드 없음)
- `.env`, `*.key`, `*.pem`, `secrets/`, `private/` 등 비밀/개인정보 파일은 `.gitignore`로 저장소에 올라가지 않습니다.

## 면책 안내
이 앱은 **문구 작성·정보 제공을 돕는 도구**이며, 법률·금융 자문이 아닙니다.
환불·위약금·계약 관련 결과를 단정하지 않으며, 실제 처리 기준은 각 업체의 약관·계약에 따릅니다.
분쟁이 해결되지 않을 경우 1372 소비자상담센터 또는 한국소비자원에 상담할 수 있습니다.
